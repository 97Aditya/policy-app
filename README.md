# Policy Management UI (Angular 16)

## Features

- Display policies in table format
- Pagination using Angular Material
- Dynamic page size options based on data
- Clean and responsive UI

## Tech Stack

- Angular 16
- Angular Material
- TypeScript

## How to Run

1. Install dependencies:
   npm install

2. Run project:
   ng serve

3. Open browser:
   http://localhost:4200

## Notes

- Data is loaded from local JSON file (assets/policies.json)
- Pagination is implemented on client side
